These are the dotfiles for my set up on the Dell XPS 15 9500 with a resolution 1920x1200.

### Info
- **OS**: Arch Linux
- **WM**: Qtile
- **Terminal**: Alacritty
- **Editor**: Neovim/VS Code
- **Compositor**: Jonaburg's fork of Picom
- **Fetch**: Neofetch
- **File Manager**: Pcmanfm
- **PDF Viewer**: zathura-git
- **Browser**: Firefox
